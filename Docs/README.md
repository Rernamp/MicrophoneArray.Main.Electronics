
# Используемая периферия
* 2 I2S синхронизированный между собой. Оба работают исключительное на приём
	* Для каждого своя роль
		* Один заводиться в режим I2S master. CK пин подаётся на микрофоны и slave
		* Второй в режиме slave. CK пин от master подаётся на slave
	* WS пины не понятно куда пока использовать. Возможно на отдельную гребёнку
	* Использовать одинаковый формат(MSB или LSB alignment standard)
* 1 TIM с возможность внешнего тактирования и выходом PWM
	* Generalpurpose timer может использоваться для этого. 
		* Настраивается тактирование с внешнего ETR пина
		* Настраивается PWM для понижения частоты
* 1 TIM с возможность внешнего тактирования
	* Любой таймер с ETR пином и **прерыванием на переполнение**
	* Заводим ETR пин на CK пин I2S master
		* Альтернативно подать на ETR пин MCK, ибо у него 256 x Fs частота = 12.288МГц
* USB
* DMA
	* нужно для I2S
* GPIO на светодиоды
