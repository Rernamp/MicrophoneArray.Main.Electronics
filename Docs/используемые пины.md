## Дополнительные примечания по сноскам из даташита:

1.  **Сноска `(1)` для пина `PA4` (I2S3_WS_1):**
    *   Это ремап-функция. Чтобы она работала, необходимо установить соответствующее значение в регистре AFIO (значение `_1`).

2.  **Сноска `(2)` для пина `PC7` (I2S3_MCK) и сноска 11:**
    *   По умолчанию работает. Если вы активируете 10M Ethernet (`ETH`), то этот пин будет занят, и функция I2S3_MCK автоматически переключится на другой пин (в даташите указан `P48`, что, вероятно, опечатка). Рекомендуется избегать этого пина при использовании Ethernet.

3.  **Сноска 12 (Конфликт SPI3_MOSI и I2S3_SD на PB5):**
    *   На `PB5` по умолчанию находится `I2S3_SD`. Однако, если вы используете Ethernet, этот пин конфликтует. В этом случае используйте `PA9` как основной пин для `I2S3_SD`.

## Рекомендации для начала разработки:

Для стандартного использования I2S3 рекомендуется следующая связка пинов (работают "из коробки" без ремапа):
*   **WS (Word Select):** `PA15` (с настройкой режима I2S) или `PA4` (с ремапом)
*   **CK (Clock):** `PB3`
*   **SD (Data):** `PA9`
*   **MCK (Master Clock):** `PC7` (если не используется Ethernet) или `PA8`

**Важно:** Для пинов с типом "Ремап" (`PA4`, `PA15`, `PC10`, `PC12`) не забудьте в коде настроить регистры переназначения периферии (AFIO), иначе они не будут работать как I2S.


> **Примечания:**
> - **MCK** используется только для I2S2 (Master Clock) и подаётся на оба аудио-кодека.
> - **I2S3** работает в режиме Slave (ведомый), тактируется от I2S2.
> - Все пины, кроме PA15, работают по умолчанию (без ремапа).

|                   | **CK** (Clock)         | **PB13**            | Default             | Короткая дорожка к кодеку                                                                                    |

## Важные примечания из даташита
### Note 14 (из PDF, стр. 14):
> `TIM2_CH1 and TIM2_ETR share the same pin, but cannot be used at the same time.`
**Перевод:** Таймерный канал 1 (TIM2_CH1) и вход внешнего триггера (TIM2_ETR) разделяют один и тот же вывод (PA0), но не могут использоваться одновременно. Вам нужно выбрать что-то одно:
- Либо **ETR** для подсчёта внешних импульсов.
- Либо **PWM** на канале CH1.
**Рекомендация:** Если вам нужен ETR, используйте для ШИМ другой канал TIM2, например, **PA1 (CH2)**.
