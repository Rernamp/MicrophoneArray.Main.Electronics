| ------------------------------ | ------------------------------ | --------------------- | -------------------- | ------------------------------------ |
| 48                             | 96                             | 64                    | 6.144                | 3.072                                |

Формула битовой частоты
CK_Fs = 32 x 2 x Fs

Необходимое значение CK_Fs = 48кГц * 64 * 2 = 6.144МГц
* Умножение на 2 потому что, для захвата по rising и faling фронту нужно в два раза чаще забирать данные
	* тактируем I2S два раза быстрее, чем частота тактирования микрофонов

Частоту тактирования микрофонов нужно понизить при помощи таймера

Из-за этого I2S нужно настраивать на частоту 96кГц(её предельное значение)
